# TDSTAMP

Marketing website for TDSTAMP – Vietnamese stamp manufacturer brand.

## Tech stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Push this repo to GitHub
2. Import the project in [Vercel](https://vercel.com)
3. Set root directory to this folder if needed
4. Optional: set `NEXT_PUBLIC_SITE_URL` to your production domain
